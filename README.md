# TeleCounter

*基於 Telegram Bot 的報數小遊戲。*

## 核心功能

- **數學運算報數**：機器人支援解析簡單的數學運算。例如當目前數字是 9 時，輸入 `5*2` 也會被視為正確的報數 `10`。
- **嚴格報數規則**：
  - **禁止連報**：同一位使用者不能連續報數兩次。
  - **按序遞增**：必須嚴格遵守 `n + 1` 的順序。
- **懲罰與重置**：
  - 如果有人報錯數字，機器人會重置報數進度至 0。
  - 報錯者會被暫時禁言（Mute），以示懲罰。
- **討論主題 (Topics) 支援**：支援 Telegram 群組的討論主題功能，可精確限制機器人在特定頻道運行。
- **資料持久化**：內建 SQLite 資料庫自動管理狀態，即使程式關閉重啟，報數也不會遺失。

## 安裝

1. 安裝 Python 3.10+
2. 安裝依賴庫：

```bash
pip install -r requirements.txt
```

## 設定

編輯 `main.py` 開頭的「配置區」以調整機器人行為：

- `TOKEN`：你的 Telegram Bot API Token。
- `ALLOWED_CHAT_ID`：限制運行的群組 ID（設為 `None` 則開放所有群組）。
- `ALLOWED_TOPIC_ID`：限制運行的 Topic ID（針對有啟用主題功能的群組）。
- `MUTE_DURATION`：報數錯誤後的禁言時間（秒）。

## 啟動

```bash
python main.py
```

## 注意事項

- 機器人必須在群組中被授予**管理員權限**（特別是「禁言成員」與「刪除訊息」權限），否則懲罰機制將失效。
- 請確保伺服器環境已安裝 `sqlite3`（通常 Python 已內建）。
