# TeleCounter

*基於 Telegram Bot 的報數小遊戲。*

## 核心功能

- **排行榜系統**：新增 `/leaderboard` 指令，可隨時查看群組中報數最高紀錄的前 10 名。
- **個人高分追蹤**：自動紀錄每位使用者的最高報數成就。
- **多樣化算式報數**：支援加減乘除、平方 (`^`) 與開根號 (`√`)，甚至支援省略乘號的寫法（如 `2(3+2)`）。
- **對話式解析**：不需要只傳送純數字，機器人會自動從訊息中抓取數字。例如：輸入「我報 10 號」或「這題答案是 1+1」皆可運作。
- **即時狀態回饋**：報數正確會得到 👍 反應，報錯則會得到 👎 並執行懲罰。
- **嚴格遊戲規則**：
  - **禁止連續報數**：同一位使用者不能連續報數，必須與他人輪流進行。
  - **精確遞增順序**：必須嚴格遵守數字加一的順序。
  - **支援小數捨去**：報數結果不一定要是整數，機器人會對所有計算結果進行「無條件捨去」（Floor）。例如輸入 `√2` 會被視為 `1`。
- **報錯懲罰機制**：一旦有人報錯，目前的報數進度將歸零重來，且報錯者會被暫時禁言。
- **支援群組討論主題**：可限制機器人僅在群組內的特定討論主題（Topic）中運行。

## 安裝

1. 安裝 Python 3.10+
2. 安裝依賴庫：

```bash
pip install -r requirements.txt
```

## 設定

編輯 `main.py` 開頭的「配置區」以調整機器人行為：

- `TOKEN`：你的 Telegram Bot API Token。
- `ALLOWED_CHAT_ID`：限制運行的群組 ID（設為 `None` 則開放所有群組）。
- `ALLOWED_TOPIC_ID`：限制運行的 Topic ID（針對有啟用主題功能的群組）。
- `MUTE_DURATION`：報數錯誤後的禁言時間（秒）。

## 啟動

```bash
python main.py
```

## 注意事項

- 機器人必須在群組中被授予**管理員權限**（特別是「禁言成員」與「刪除訊息」權限），否則懲罰機制將失效。
- 專案已改用異步資料庫，請確保 `requirements.txt` 中的 `aiosqlite` 已正確安裝。
